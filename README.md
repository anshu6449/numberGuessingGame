# 🎯 Number Guessing Game – Python

## 📌 Project Description
The Number Guessing Game is a simple command-line Python application where the computer randomly selects a number within a given range, and the player attempts to guess it. The program provides feedback after each guess, indicating whether the guess is too high or too low until the correct number is guessed.

This project demonstrates basic Python programming concepts such as loops, conditional statements, user input handling, and random number generation.

## 🚀 Features
- Random number generation using Python's `random` module
- Interactive command-line gameplay
- Feedback on each guess (Too High / Too Low)
- Counts the number of attempts

## 🛠️ Technologies Used
- Python
- Random module
